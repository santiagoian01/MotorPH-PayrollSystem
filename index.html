Introduction

This section provides users with a brief overview of the payroll system, identifies the intended audience, and offers instructions on how to effectively navigate and utilize the user guide. The MotorPH Payroll System is a desktop-based application tailored for HR operations, streamlining payroll processing, leave management, and employee data management with security and efficiency.

Getting Started

  • Hardware/Software Requirements: Java Runtime Environment (JRE), a Java IDE (e.g., IntelliJ IDEA, Eclipse), and access to a Windows-based machine.
  • Accessing the System: Clone the repository from GitHub, compile using the IDE or command line, and launch the application.

Using the Payroll System

  1. Secure Login Procedures: Login using valid username and password credentials specific to each role (HR Manager, Payroll Staff, Employee).
  2. Salary and Deduction Calculations: Automated salary computation based on employee records, deductions, and applicable benefits.
  3. Leave Applications: Employees can submit leave requests; HR can approve or reject them directly in the system interface.
