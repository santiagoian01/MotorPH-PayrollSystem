Project Overview

The MotorPH Payroll System is a desktop-based application tailored for HR operations at MotorPH. It provides functionalities for:

  • Secure Login for employees and HR
  • Leave Request Management
  • Employee Data Management (CRUD)
  • Accurate Payroll Processing with Deduction Calculations

Features

  1. Secure Login: User authentication with error handling.
  2. Employee Leave Management: Submit/view leave requests; HR approval/rejection.
  3. HR Manager Duties: CRUD for employee records; manage leave requests.
  4. Payroll Processing: Calculate salaries with deductions; generate/store payslips.

How to Use

  1. Clone the repository from GitHub.
  2. Compile the project using any Java IDE or command line.
  3. Launch the application and log in with valid credentials.
  4. Access respective modules: Leave, Employee Management, Payroll.
