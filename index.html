MotorPH Payroll System

Developed by Group 12

About the Project

The MotorPH Payroll System is a Java-based application designed to streamline payroll processing, employee leave management, and HR operations for MOTORPH. This system ensures secure, accurate, and efficient handling of employee data and payroll records.
